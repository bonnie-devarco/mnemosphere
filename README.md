# Mnemosphere

Mnemosphere is a living digital memory palace for symbolic form, design science,
cultural memory, and connective intelligence.

This GitHub Pages site is intentionally simple: static HTML pages with one
shared CSS file, no JavaScript, no build step, and no external dependencies.

Pages:

- `index.html`
- `about.html`
- `connective-intelligence.html`
- `writings.html`
- `presentations.html`
- `archive.html`
- `links.html`
